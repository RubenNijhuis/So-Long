### Installation
This project assumes you have Make installed.

```bash
# Clone:
$ git clone https://github.com/RubenNijhuis/so-long.git

# Installing all sub-packages
$ git submodule update --init --recursive

# Compile:
$ Make all
```

### Running the game
```bash
# Run:
$ Make run
```
